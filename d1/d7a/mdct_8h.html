Compute inverse MDCT of size N = 2^nbits

Parameters
outputN samples
inputN/2 samples "In-place" processing can be achieved provided that: [0 .. N/2-1 | N/2 .. N-1 ] <-—input-—> <--------—output--------—>

The result of ff_imdct_half is to put the 'half' imdct here 

                     N/2          N-1
                     <--half imdct-->

We want it here for the full imdct: N/4 3N/4-1 <-----------—>

In addition we need to apply two symmetries to get the full imdct: 

      <AAAAAA>                <DDDDDD>
              <BBBBBB><CCCCCC>

      D is a reflection of C
      A is a reflection of B (but with sign flipped)

We process the symmetries at the same time as we 'move' the half imdct from [N/2,N-1] to [N/4,3N/4-1]
